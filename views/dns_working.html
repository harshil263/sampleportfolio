How DNS server working?

Description

  • DNS stand for Domain name server or name.
  • What is does is get the domain name address and find the corresponding IP address associated with it.
  • Basically DNS is a standard set of protocol for how computer exchange the data on internet and many private networks. Also known as a TCP/IP.
  • Computer and other networks use IP address to route your request to serve the request.
  • Once we request the any web site then computer will look all DNS server recursively and find the machine readable IP address.
  • Once IP address found computer will direct that IP address over a network and request the response.
  • While searching for the correct IP address DNS recursive servers will cache this IP address so in future if same request generate then it will serve from cache.
  • So this way request will save the serving time and response will become faster.
