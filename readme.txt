=> Steps to run the index.html.

1) Copy the folder to any location in your machine.
2) Run the index.html file.
	
	a) This file has all question's link.
	b) Click on each link and get the answer by giving values.
	
3) There is HTML file name ""all_answers.html". This file you can directly run in browser.
4) All images are responsive with respect to the viewport width.
5) One css file is applicable to all HTML pages.
